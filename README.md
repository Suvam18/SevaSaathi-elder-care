-----------Description-------------

SevaSaathi is a web-based platform designed to help elderly individuals manage their daily healthcare activities easily. Many senior citizens face difficulties remembering medications, keeping track of doctor appointments, and accessing assistance when needed.

SevaSaathi provides a simple digital solution that allows elderly users to set medication reminders, schedule doctor appointments, and connect with caregivers or volunteers for support.

The platform focuses on improving the independence, safety, and quality of life of elderly people while also helping families and caregivers stay connected with them.

-------------Architecture Overview-----------------

The architecture of SevaSaathi follows a simple client-based web application structure.

Client Layer

User interface accessible through web browsers

Allows elderly users to interact with features such as reminders and appointment management

Application Layer

Handles the core logic of the platform

Processes reminder scheduling, user inputs, and UI updates

Data Layer

Manages the storage of reminder data, appointment schedules, and user preferences
